Checkpoint workflow solutions blend step-by-step guidance with easy-to-use graphical workflows and tools in common but complex areas. To provide best practice processes, you can link directly to related commentary, calculators, checklists, flowcharts, form and templates.

Each Workflow Solution has two main components:

  • Workflow Navigation Map:
    Workflows provide guidance on a subject area, where items are broken down step by step. Each item provides a more detailed graphical outline of the key components from which the user can link to relevant sources such as legislation, commentary, news or interactive tools.
  • Workflow tools
    The Tools provides a variety of time-saving online tools to help you find and organise information and forecast and manage business scenarios. You have the ability to save tools, export tools and customise them for specific clients.

We have designed workflows to be easy to use. There are several supporting features which will aid in navigating around workflow.

These features are made available once you click through to a navigation schematic or workflow tool and include:

  • Document in Context
    Shows the relative location of the document in the Workflow Table of Contents
  • Workflow Tools
    Provides direct access to all the tools referenced in the current document
  • Related Documents
    Provides direct access to all the sources which include commentary, legislation and news items that are referenced in the current document
  • Recently accessed Documents
    Provides an audit trail of the last 6 documents accessed in the site.

Icons are used extensively in our workflow diagrams. Below is an explanation of their meaning.

Icon Name When it's used
Tool Indicates a link to a relevant tool in the 'tools' tab.
Tip Highlights a useful or practical tip.
Trap Warns about a tricky, unintended or unforeseeable aspect.
Alert Warns to be aware of a specific situation in regard to the workflow
Commentary Links to relevant commentary in the 'research' tab.
Definition Indicates that the information displayed is a definition.
See Also Highlights a link to another step in the workflow.
